# Youpurchase - 酒厂管理员manager端

## 文件组织结构

components为路由跳转的根目录

js文件所在位置表明了该组件的路由跳转目录(若子文件夹与当前js文件同名则代表子文件夹为该组件的子组件文件夹)

## 功能分配

app负责控制用户名信息

homepage为登录页面

shopManage和dealerManage分别为店铺管理及经销商管理
